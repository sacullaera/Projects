# Projeto de Banco de Dados – Avaliação N2 (Fatec)

Sistema para **gestão de ativos e vulnerabilidades** com:
- RBAC (Role-Based Access Control)
- Auditoria automática via triggers (UPDATE/DELETE)
- Dashboard de vulnerabilidades tratadas vs. não tratadas
- Modelagem relacional completa (1:N e N:N)

## Tecnologias
- SQLite3
- SQL (DDL, DML, Views, Triggers)
- Draw.io (modelo lógico)

## Arquivos
- `n2_lucas_areal.sql`: script completo de criação, população e triggers
- `n2_lucas_areal_modelo_logico.drawio`: modelo lógico/relacional
- `n2_lucas_areal.sqbpro`: projeto do SQLite3
- `n2_lucas_areal_DB.db`: banco de dados do projeto

> **Projeto acadêmico – destinado à avaliação da disciplina de Banco de Dados (2025/2).**
